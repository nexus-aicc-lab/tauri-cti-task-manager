📋 설치 방법

  1. 위의 "직접 다운로드" 버튼을 클릭하여 설치 파일을 다운로드합니다.
  2. 다운로드된 CTI_Task_Master_Setup.exe 파일을 실행합니다.
  3. 설치 마법사의 안내에 따라 프로그램을 설치합니다.
  4. 설치 완료 후 "웹 런처 열기"를 클릭하여 로그인합니다.
  5. 로그인 정보를 입력하고 "프로그램 실행 및 로그인" 버튼을 클릭합니다.
